True Clock is a SMAPI mod for Stardew Valley that shows your real local time on the in-game HUD. It also supports up to five configurable real-time alerts with a clock balloon above the player, a small HUD message, and a short alarm sound.

The clock uses your computer's local time, not Stardew Valley's in-game time.

## Features

- Shows the real local time below the top-left status area.
- Uses Stardew Valley's native UI style.
- Supports 24-hour or AM/PM display.
- Supports up to five configurable alerts.
- Each alert can be enabled or disabled independently.
- Each alert has an hour, minute, and optional message.
- Alerts trigger at most once per calendar day.
- Alerts last up to five seconds and do not pause the game or block input.
- Optional Generic Mod Config Menu support.

## Requirements

- Stardew Valley
- SMAPI 4.0.0 or newer
- .NET 6 SDK only if you want to build the mod from source
- Generic Mod Config Menu is optional

## Installation From Nexus Mods

True Clock can also be installed from Nexus Mods once the mod page is published.

- Nexus Mods Stardew Valley: https://www.nexusmods.com/stardewvalley
- Mod page: to be added after the Nexus Mods ID is available

Download the mod archive from Nexus Mods, then extract it into your Stardew Valley `Mods` folder.

## Installation From Release ZIP

1. Install SMAPI from https://smapi.io/.
2. Download the latest `TrueClock.zip` from the GitHub Releases page.
3. Extract the ZIP into your Stardew Valley `Mods` folder.
4. Make sure the final folder looks like this:

```text
Stardew Valley/
  Mods/
    TrueClock/
      manifest.json
      TrueClock.dll
      i18n/
```

5. Start the game through SMAPI.

On first launch, the mod creates `config.json` inside the `TrueClock` mod folder.

## Configuration

If Generic Mod Config Menu is installed, open the mod settings in-game and configure the clock and alerts there.

Without Generic Mod Config Menu, edit `config.json` manually after the first launch. The mod always keeps exactly five alert slots.

Example:

```json
{
  "Use24HourClock": true,
  "Alerts": [
    {
      "Enabled": true,
      "Hour": 8,
      "Minute": 30,
      "Message": "Time to check the farm"
    }
  ]
}
```

Hours use real local time in 24-hour format from `0` to `23`. Minutes use `0` to `59`.

## Building From Source

Install the .NET 6 SDK, then run:

```bash
dotnet build -c Release
```

The release workflow in this repository builds the mod and publishes a ZIP file for GitHub Releases.

## GitHub Release ZIP

Official releases are generated by GitHub Actions. When a version tag like `v1.0.0` is pushed, the workflow builds the mod, creates `TrueClock.zip`, and attaches it to the GitHub Release.
